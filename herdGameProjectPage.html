I created a game with the Unity game engine. I have recently finished Unity's Essentials Pathway, and I am now working on their Programming Basics Pathway. This is one of the games that I created with the knowledge I have gained from the courses. The object of the game is simple, move left and right to try to avoid the herd of animals coming towards you. You can also press space, to shoot pizza at the herd to make them disappear. If an animal touches the character, the character dies.

This game is far from perfect, it is still missing a lot of features. For example, the game doesn't currently have a Main menu, and it also doesn't have a "Game Over" screen to replay the game after you die, you have to refresh the browser.

There are lots of other things that I would change if I were to do this project again. For starters, I would make it a lot more difficult. Currently, the animals that you are supposed to avoid, spawn too infrequently, the game would be a much more fun experience if it was just a bit harder. I would also add music and sound effects if I were to do this again. Sound seems like a simple thing, but it can make a huge difference.
